|---------------------+------------------------------------------------------------------------------+---|
| Using Stencil Block | Stencil test make Buffer holding a refV. if new pixel meet                   |   |
|                     | [[CompFN]] (return t), new Pixel will be written into Buffer,                    |   |
|                     | and use [[OP]] to make some change on refV, else do nothing(that is transparent) |   |
|                     | using Stencil{..} block in Subshader is enough to enable stencil test        |   |
|---------------------+------------------------------------------------------------------------------+---|
| Ref                 | refV:BYTE                                                                    |   |
|---------------------+------------------------------------------------------------------------------+---|
| ReadMask, WriteMask | v:BYTE                                                                       |   |
|---------------------+------------------------------------------------------------------------------+---|
| Comp                | [[CompFN]]                                                                       |   |
|---------------------+------------------------------------------------------------------------------+---|
| Pass,Fail,ZFail     | [[OP]]                                                                           |   |
|---------------------+------------------------------------------------------------------------------+---|
| <<CompFN>>          |                                                                              |   |
|---------------------+------------------------------------------------------------------------------+---|
| Equal               |                                                                              |   |
|---------------------+------------------------------------------------------------------------------+---|
| NotEqual            |                                                                              |   |
|---------------------+------------------------------------------------------------------------------+---|
| Always              |                                                                              |   |
|---------------------+------------------------------------------------------------------------------+---|
| Never               |                                                                              |   |
|---------------------+------------------------------------------------------------------------------+---|
| <<OP>>              |                                                                              |   |
|---------------------+------------------------------------------------------------------------------+---|
| Keep,Replace        |                                                                              |   |
|---------------------+------------------------------------------------------------------------------+---|
| IncrSat,DecrSat     | inc/dec refV until 255/0 and stop                                            |   |
|---------------------+------------------------------------------------------------------------------+---|
| IncrWrap,DecrWrap   | inc/dec refV until 255/0 and round back                                      |   |
|---------------------+------------------------------------------------------------------------------+---|
| Invert              | !refV                                                                        |   |

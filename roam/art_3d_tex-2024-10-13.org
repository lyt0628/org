#+NAME: Texture Type
|----------+---+--------------------------------------------------------------------------------------------|
| Texure   |   |                                                                                            |
|----------+---+--------------------------------------------------------------------------------------------|
| Diffuse  |   | 固有色                                                                                     |
|----------+---+--------------------------------------------------------------------------------------------|
| Matal    |   | 金属度, 反映金属特性,体现为高光更清晰(反射颜色接近固有色),对漫反射不敏感                   |
|          |   | 注意: 光泽度表示对光线的镜像反射的敏感程度,体现为高光更清晰,整体更亮和反射呈像更清晰(镜子) |
|----------+---+--------------------------------------------------------------------------------------------|
| Ao       |   | 模型自身遮挡产生的光影效果                                                                 |
|----------+---+--------------------------------------------------------------------------------------------|
| 法向     |   | 体现物体自身的凹凸,以此影响对光线反射的计算                                                |
|----------+---+--------------------------------------------------------------------------------------------|
| Emission |   | 发光贴图, 黑色表示不发光, 其他颜色发光                                                     |
|----------+---+--------------------------------------------------------------------------------------------|
|          |   |                                                                                            |



- 大部分贴图作用都是用于弥补低模的不真实性,在高模不会有这种缺陷(Ao, 法向, 高度)
- 贴图怎么上Mesh, 取决于 UV坐标, 在不同软件 UV坐标 都是兼容的, 即直接把贴图复制过去就直接可以上在正确的位置
- 一个物体拥有一个材质队列, 这个材质队列的下标在不同软件之间也是兼容的
- 注意, 按显示的性质选择贴图方式, 不要全部的颜色都放到固有色贴图
- 把网格进行合并,然后指定不同下标的材质, 比较好维护
- 特别注意, 阴影 要放在 Ao 贴图, 尽管没有阴影的 Diffuse 贴图不怎么好看
- 多个材质按顺序写进 Buffer, 后面的 Material 如果是透明的, 那么透明的区域就会显示上一层的材质, 这个特性用来做贴花(decal), 性能影响较大

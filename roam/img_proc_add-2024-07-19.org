|-----------------+-----------------------------------------------+---+----------------------+---------------|
| Method          | equalation                                    |   | Note                 | Chinese       |
|-----------------+-----------------------------------------------+---+----------------------+---------------|
| Linear Blending | final = img1.rgb*factor + img2.rgb*(1-factor) |   | img1 is lower image. | 线性混合      |
|                 |                                               |   | img2 is top image    |               |
|-----------------+-----------------------------------------------+---+----------------------+---------------|
| Multipky        | final = img1.rgb*img2.rgb                     |   |                      | 正片叠底,乘算 |
|-----------------+-----------------------------------------------+---+----------------------+---------------|
| Screen          | final = 1 - [(1-img1.rgb) * (1-img2.rgb)]     |   |                      | 滤色          |
|-----------------+-----------------------------------------------+---+----------------------+---------------|
| Overlay         | if img1.value < 0.5                           |   |                      | 叠加          |
|                 | final = 2 * img1.rgb * img2.rgb               |   |                      |               |
|                 | else                                          |   |                      |               |
|                 | final = 1 - 2*(1-img1)*(1-img2)               |   |                      |               |
|-----------------+-----------------------------------------------+---+----------------------+---------------|
| Soft Light      | if img1.value < 0.5                           |   |                      | 柔光          |
|                 | final = img1*[1 - (1- 2*img2)*(1-img1)]       |   |                      |               |
|                 | else                                          |   |                      |               |
|                 | final = img1*(2*img2 -1)*(img1*img1 - im1)    |   |                      |               |
|-----------------+-----------------------------------------------+---+----------------------+---------------|
| Hard Light      |                                               |   |                      | 强光          |
|-----------------+-----------------------------------------------+---+----------------------+---------------|
| Difference      |                                               |   |                      | 差值          |
|-----------------+-----------------------------------------------+---+----------------------+---------------|
| Exclusion       |                                               |   |                      | 排除          |

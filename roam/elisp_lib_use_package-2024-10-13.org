|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| Keyword             | Params                                            | Ret | Note                                                                 |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :init               | List                                              |     | List that will be run before package is loaded                       |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :config             | List                                              |     | List that will be run after package is loaded                        |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :commands           | List                                              |     | List of interactive command name that defined by the package         |
|                     | or a single Symbol                                |     | those command will be autoload                                       |
|                     |                                                   |     | usually used to begine a major or minor mode                         |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :bind               | List of Pair(cons)                                |     | Each Key is Key String, value is the command  will be bound          |
|                     |                                                   |     | those command will be autoload                                       |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
|                     | A single cons                                     |     |                                                                      |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
|                     | ("Keystring" ("Desc" . <Command>))                |     |                                                                      |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
|                     |                                                   |     | :bind ([remap fill-paragraph] . unfill-toggle)                       |
|                     |                                                   |     | will bind unfill-paragraph to M-q(original fil-pragraph)             |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :bind(              |                                                   |     | Bind Command to a local keymap                                       |
| :map <a local map>  | List of cons                                      |     | can redeclare :map modifier to use other keymap                      |
| <params>)           |                                                   |     |                                                                      |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :auto-load          | List                                              |     | List of non-interactive function that will be autoload               |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :bind-keympa        | List of cons                                      |     | Bind a keymap to Keystring work as :bind                             |
|                     | or a single cons                                  |     | notice: keymap must use it to bind instead of :bind                  |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :mode               | regx string for extension like "\\.rb\\"          |     | spec extension for interpreter lang (work with [[:interpreter]])         |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| <<:interpreter>>    | string of interpreter                             |     | spec interpreter                                                     |
|                     | or cons ("interpreter name" . <interpreter-mode>) |     | default interpreter-mode is name of package.                         |
|                     |                                                   |     | specially for package python, name of mode is 'python-mode'          |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :defer t            | t                                                 |     | enable defer load but you must autoload command by autoload function |
|                     |                                                   |     | the keyword of use-package shound not be used                        |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :hook               | cons of target-hook=>function                     |     | the same as (add-hook 'target-mode-hook #'function)                  |
|                     | or list of conses                                 |     | also, the symbol of function will be autoload                        |
|                     |                                                   |     | keyword :hook is re-enter-able                                       |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :custom             | list of (custom-variable value "desctring")       |     |                                                                      |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :custom-face        |                                                   |     |                                                                      |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :if                 | bool expresion                                    |     | if t, tge use-package statement can be used                          |
|                     |                                                   |     | eg. window-system                                                    |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :disabled  or :when |                                                   |     | tun off evalation                                                    |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
| :unless             |                                                   |     | eq to :if (not <expression>)                                         |
|---------------------+---------------------------------------------------+-----+----------------------------------------------------------------------|
|                     |                                                   |     |                                                                      |

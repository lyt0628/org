|-------------+-----------+---------------------------------------------------------------------------------|
| Propery     | Accessor  | Note                                                                            |
|-------------+-----------+---------------------------------------------------------------------------------|
| pid : pid_t | getpid()  | process ID                                                                      |
|-------------+-----------+---------------------------------------------------------------------------------|
| ppid        | getppid() | parent process ID                                                               |
|-------------+-----------+---------------------------------------------------------------------------------|
| uid         | getuid()  | user ID                                                                         |
|             |           | this is RUID, real user ID                                                      |
|-------------+-----------+---------------------------------------------------------------------------------|
| euid        | geteuid() | generaly is uid,                                                                |
|             |           | when command sudo su was executed, auth become super. euid will be root user ID |
|-------------+-----------+---------------------------------------------------------------------------------|
| gid         | getgid()  | group id  of user thatcreate the proc                                           |
|-------------+-----------+---------------------------------------------------------------------------------|
| egid        | getegid() | like euid                                                                       |
|-------------+-----------+---------------------------------------------------------------------------------|
| sid         | getsid()  | process session ID                                                              |
